## 📅 每日详细作战路线

### D1（5月1日）：高铁穿梭，轻松挺进深山古镇
**📍 核心任务：丝滑接驳，将长途疲劳降至最低。**

* **09:00 - 11:30【高铁时光】** 乘坐南京南 ➔ 衢州站 / 衢州西站。孩子们在车厢内自由走动，告别安全座椅的长时间束缚。
* **11:30 - 12:30【落地提车】** 抵达衢州。直接在高铁站停车场提取提前订好的车辆，行李扔进后备箱。
* **12:30 - 13:30【市区午餐】** 在高铁站附近吃个便饭。
* **13:30 - 15:00【南下入山】** 衢州市区 ➔ 廿八都古镇（约1.5小时）。平稳的省道，刚好让孩子们在车上睡个午觉。
* **15:00 - 15:30【安营扎寨】** 抵达**廿八都名都酒店**（北入口游客中心旁，带大型停车场），办理入住。
* **15:30 - 17:30【推车漫游】** （*凭提前抢好的门票入园*）推车逛古镇主街，看各省门楼，感受江湖气息。
* **17:30 - 20:00【古镇夜色】** 晚餐后看红灯笼亮起，拍全家福，早早回房休息。

### D2（5月2日）：晨雾古镇与世遗野餐，移师大本营
**📍 核心任务：避开景交车排队，解锁“草坪山丘”遛娃宝地。**

* **07:30 - 09:00【独享清晨】** 趁大巴团未至，带家人逛空无一人的古镇，买点热腾腾的铜锣糕。
* **08:30【准时拔营】** 办理退房，驱车前往江郎山（**务必早出发，抢第一波景交车**）。
* **09:30 - 13:00【江郎山轻松游】** （*凭4月25日抢好的套票*）
    * 坐景交车直达核心区。
    * 推车走**须女湖**平缓木栈道看水。
    * 在江郎山**大草坪**铺野餐垫，吃零食，远观“三爿石”拍全家福（**坚决不爬台阶**）。
* **13:00 - 14:30【午休转移】** 江郎山 ➔ 衢州市区。完美的1.5小时车程，孩子们在车上熟睡。
* **14:30 - 15:30【入驻大本营】** 抵达**衢州西站体育中心亚朵酒店**，办理入住，彻底卸下所有行李。
* **16:00 - 18:30【草坪放风】** 步行2分钟直达**衢州体育中心**。让孩子们在起伏的绿色草坡上尽情奔跑打滚。
* **19:00【市区晚餐】** 酒店附近享用晚餐，回房回血。

### D3（5月3日）：市中心人文慢游（全程纯玩不开车）
**📍 核心任务：恐龙探秘与南孔儒风，零压力城市度假。**

* **09:00 - 10:30【史前对话】** 打车前往**衢州市博物馆**，直奔一楼参观“礼贤江山龙”骨架化石。
* **10:30 - 12:00【科学启蒙】** 步行5分钟至**衢州科技馆**，体验互动小实验。
* **12:00 - 14:30【黄金午休】** 酒店附近午餐，**直接回亚朵房间睡个高质量长午觉**。
* **15:00 - 16:30【南孔儒风】** 打车前往**孔氏南宗家庙**，在清幽的后花园喂锦鲤。
* **17:00 - 20:30【衢江烟火】** 步行或打车至**水亭门**（*官方确认免预约*）。
    * 登城墙看衢江绝美日落。
    * 晚餐吃“古铺良食”（有极度友好的不辣儿童餐）。
    * 看灯光秀（约19:00/20:00），一站式买齐带回南京的烤饼、麻糍等特产。
    * **💡 避坑提示：** 水亭门节假日停车极难，建议全程打车往返。

### D4（5月4日）：东线探秘与古建寻宝
**📍 核心任务：错峰下地宫，下午利用大本营周边休闲。**

* **08:00 - 08:45【高速驰骋】** 酒店早餐后驱车前往龙游（全程高速约40分钟）。
* **08:45 - 11:30【地下奇观】** （*凭5月1日抢好的门票*）抢第一批刷入**龙游石窟**。洞内**16℃**，务必给孩子带薄外套和小手电筒。
* **11:30 - 13:00【本地风味】** 龙游县城吃午餐（点单强调全桌免辣，来份发糕和葱花馒头）。
* **13:30 - 16:00【古建寻宝】** 开车15分钟至**龙游民居苑**。发布预先准备的“寻宝图”，让孩子找木雕小狮子。
* **16:00 - 16:50【安全撤退】** 开车回亚朵酒店。
* **17:00 - 19:00【休息回血】** 再次去体育中心草坪放风，或利用酒店洗衣房清理衣物。
* **19:00【市井夜宵】** 打车去“马站底”美食街，体验最地道的衢州小吃。

### D5（5月5日）：睡到自然醒，丝滑退房还车
**📍 核心任务：优雅告别高速大军。**

* **08:30 - 10:30【从容收拾】** 睡到自然醒，享用酒店营养早餐，慢慢将特产和行李打包。
* **10:30 - 11:30【还车进站】** 办理退房，驱车前往高铁站指定地点还车，轻松进站。
* **12:00之后【平稳回家】** 乘坐高铁返回南京。当别人正堵在跨省高速上焦虑时，你们全家可以喝着饮料，2.5小时后顺利闪现南京。
